# Amazon 商品价格监控工具 — 小白也能用

> 你不需要会写代码。你只需要会跟 AI 聊天。

这个工具能帮你自动盯住 Amazon 上任何商品的价格、评分、评论数和库存变化。**采集的是 Amazon 页面上的真实数据，绝不造假。**

---

## 我怎么安装？

**一句话就够了。** 打开你用的 AI 工具，对它说下面这句话：

### 如果你用的是 WorkBuddy

> 「帮我安装 Amazon ASIN 监控」

或者也可以说：

> 「我想监控 Amazon 商品价格，帮我搭一个」

AI 会自动帮你完成所有安装。装好后会在浏览器里打开一个图表页面。

### 如果你用的是 Claude Desktop

> 「帮我安装这个 GitHub 项目：https://github.com/CookieCN/amazon-asin-monitor」

Claude 会告诉你需要先装 Python，然后配置一个 MCP 连接。跟着它说的做就行。

### 如果你用的是 Cline（VS Code 插件）

跟 Claude Desktop 一样，对 Cline 说：

> 「安装 https://github.com/CookieCN/amazon-asin-monitor，配置成 MCP server」

### 如果你没有任何 AI 工具

那你需要一个懂一点电脑的朋友帮你装。把下面这个发给 ta：

```
1. 确保电脑装了 Python（python.org 下载）
2. 打开终端/命令提示符，输入：
   pip install requests beautifulsoup4 lxml
3. 下载项目文件到电脑上
4. 双击运行 server.py
5. 浏览器打开 http://localhost:8932
```

## 装好之后怎么用？

装好后，AI 会自动帮你打开一个网页（`http://localhost:8932`），这就是你的监控仪表盘。

### 添加你想监控的商品

在 Amazon 上找到你想盯的商品，复制它的 ASIN 编号。

> 什么是 ASIN？Amazon 商品页面的网址里 `dp/` 后面那串字母数字就是 ASIN，比如 `B0FKHC8PPV`。

然后打开你的仪表盘页面，在左侧边栏点击「添加 ASIN」，粘贴进去就行。

或者直接对 AI 说：

> 「帮我监控 ASIN B0XXXXXXX」

### 查看数据

- **价格走势图**：红色上涨，绿色下跌
- **评分变化**：看看商品口碑有没有波动
- **评论增长**：判断销量热度
- **库存状态**：有没有断货

页面右上角可以切换查看 7 天 / 30 天 / 90 天 / 全部数据。

### 让 AI 每天自动帮你盯

对 AI 说：

> 「每天早上 9:30 和晚上 6:30 帮我自动采集一次 Amazon 数据」

AI 会帮你设置定时任务。**注意：你的 AI 工具必须保持运行，定时任务才能执行。**

## 常见问题

### 数据采集不到，显示「--」怎么办？

这是 Amazon 的反爬虫机制在起作用。告诉 AI：

> 「清除 Cookie 然后重试采集」

或者等几个小时再试。大部分时候第一次采集会比较慢（工具需要和 Amazon 建立「信任关系」），之后就会顺利很多。

### 我想监控日本 Amazon 的商品

加商品时对 AI 说：

> 「添加日本 Amazon 的 ASIN B0XXXXXXX」

默认是美国 Amazon，工具也支持日本、英国、德国等 8 个站点。

### 能同时监控多少个商品？

理论无上限，但建议控制在 20 个以内。多了容易被 Amazon 限制。

### 数据准确吗？

数据直接从 Amazon 产品页抓取，不经过任何中间处理。如果抓取失败，会显示 `--`，不会编造假数据。

## 技术说明（给愿意了解的朋友）

这个工具的核心是一个 Python 爬虫，用了 7 层反检测技术来避免被 Amazon 拦截：

1. 每次请求换不同的浏览器身份（30+ 种）
2. 保存 Cookie，假装是同一个人反复来访
3. 先访问 Amazon 首页「热身」，再去看商品
4. 模拟从 Google 搜索点进来的行为
5. 请求之间随机等待几秒（像真人浏览）
6. 多种方式提取数据，一种不行换另一种
7. 被拦截后自动退避重试

这些手段让爬虫看起来更像一个真人，而不是机器人。
